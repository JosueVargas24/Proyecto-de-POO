### Características
* **Autenticación por Roles:** Control de acceso diferenciado para Administrador, Estudiante e Invitado.
* **Seguridad:** Cifrado de contraseñas mediante algoritmos de hashing.
* **Operaciones CRUD:** Creación, lectura, actualización y eliminación de registros de implementos deportivos.
* **Validaciones Robustas:** Control de campos obligatorios, límites de caracteres, restricciones en formatos de correo (@) y contraseñas seguras.
* **Persistencia Local:** Gestión de datos mediante MySQL (vía XAMPP).

### Tecnologías y Entorno
* **Lenguaje:** Java (JDK 24 - 26)
* **Interfaz Gráfica:** JavaFX SDK
* **Base de Datos:** MySQL / XAMPP
* **IDE Recomendado:** IntelliJ IDEA
